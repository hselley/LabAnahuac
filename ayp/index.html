Acceso por VNC a lab.anahuac.mx

Se puede entrar al servidor de tres maneras:

  1. Mediante un cliente de SSH y otro de VNC (Virtual Network Computing), por ejemplo Putty y vncviewer
  2. Mediante Guacamole. (Antes de emplear este sistema, por favor lea el documento de como usar Guacamole en lab.anahuac.mx)
  3. Guacamole y Gambas con sonido

Guacamole es una opción muy cómoda y simple de utilizar, por ello es la más recomendable. A continuación se mencionan algunas ventajas y deventajas de usarlo.

Ventajas de emplear Guacamole:

  • Sólo se necesita un navegador que soporte HTML5 (Mozilla Firefox, Google Chrome o Safari).
  • Las conexiones de SSH y VNC son dentro del mismo servidor por lo que son mucho más estables. Si hay un fallo en la red, el sistema Guacamole se vuelve a reconectar automáticamente a las sesiones abiertas.
  • No es necesario configurar un protocolo, túnel, usuario, contraseña, puerto, etc. Todo esto ya está configurado previamente en el servidor.
  • Se puede usar incluso dispositivos móviles como celulares o tabletas. Es recomendable usar navegadores web sencillos, como el DuckduckGo, Yuzu Browser, Pure Browser, etc. En realidad prácticamente cualquier navegador sirve.

Desventajas de emplear Guacamole:

  • La funcionalidad de las teclas depende del navegador que se emplee. Para las teclas con acentos se puede usar Chromium por ejemplo.
  • El copiar y pegar se tiene que hacer mediante una ventana intermedia del navegador, no se puede hacer directamente.
  • Es importante tener cuidado con la seguridad de las sesiones ya que al ser más facil de entrar y mantenerse abiertas, sino se cierran explícitamente alguien más puede usarlas indebidamente.
  • Por el momento si se actualiza alguna contraseña del usuario, ya no es posible entrar al sistema por Guacamole ya que es necesario solicitar al administrador del sistema que actualice los registros de Guacamole. Es decir los cambios de contraseña no se ven reflejados automáticamente en el sistema Guacamole.

Más información de como usar Guacamole, por ejemplo copia y pegar, movimiento del ratón, teclado, etc.

Nota: no todas las funcionalidades, como por ejemplo la transferencia de archivos, el sonido, están instaladas en el servidor.
